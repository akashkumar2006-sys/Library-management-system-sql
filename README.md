# Library-management-system-sql
Library management system using command of sql
Library Management System

A simple SQL-based Library Management System project developed using MySQL.
This project demonstrates the use of relational databases, table relationships, and SQL queries for managing library records.

Features

- Store student information
- Store book details
- Manage issued books
- Track returned and non-returned books
- Perform data retrieval using SQL queries

Technologies Used

- SQL
- MySQL

Database Tables

Students

Contains student details such as:

- Student ID
- Student Name
- Department
- Year of Study

Books

Contains book details such as:

- Book ID
- Book Name
- Author Name
- Available Copies

IssuedBooks

Maintains records of:

- Issued books
- Issue date
- Return date

SQL Concepts Used

- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- INSERT INTO
- SELECT Queries
- WHERE Clause
- GROUP BY
- JOIN Operations
- Aggregate Functions
- UPDATE
- DELETE

Sample Queries

- View all students
- View available books
- Find books not returned
- Count books issued by each student
- Retrieve issued book details using JOIN

Project Objective

The objective of this project is to understand database management concepts and improve SQL query writing skills through a practical implementation.
